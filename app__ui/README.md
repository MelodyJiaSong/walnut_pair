# Walnut Pair UI Application

Frontend application built with Next.js, React, TypeScript, Redux, and Ant Design.

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
